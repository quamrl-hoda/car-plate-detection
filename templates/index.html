Pipeline Stages

    

      

        01
        
Data Ingestion

        
Downloads dataset from Google Drive via gdown, extracts and normalises folder structure into YOLO train/valid/test splits.

      

      

        02
        
Data Validation

        
Verifies all required splits and label files exist. Writes a status report before allowing training to proceed.

      

      

        03
        
Data Transformation

        
Copies splits to artifacts directory and rewrites data.yaml with absolute paths so YOLO resolves them correctly.

      

      

        04
        
Model Training

        
Fine-tunes YOLOv8n on the licence plate dataset. Saves best.pt and last.pt weights tracked by DVC.

      

      

        05
        
Model Evaluation

        
Runs validation to compute mAP50, mAP50-95, precision and recall. Saves metrics.json for experiment tracking.

      

    

  


  
  

    

      

        
Retrain Model

        
Trigger a full pipeline run. All five stages will execute sequentially. This may take several minutes on CPU.
